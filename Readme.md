# 📊 Sales & Revenue Analysis Dashboard

#Project Overview
This Power BI dashboard analyzes sales and revenue data to generate meaningful business insights.
The dashboard helps visualize:
- Total Sales
- Total Profit
- Total Orders
- Total Quantity Sold
- Profit Margin
- Sales Trend Over Time
- Top 10 Products by Sales
- Sales by Region
- Profit by Sub-Category

## 📂 Dataset
Sample Superstore Dataset
## 🛠 Tools Used
- Microsoft Power BI
- Power Query
- DAX
- CSV Dataset

## 📈 Dashboard Features
- KPI Cards
- Interactive Slicers
- Line Chart
- Bar Charts
- Revenue Analysis
- Product Performance
- Regional Sales Analysis
